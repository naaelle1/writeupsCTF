# CTF Writeups

This repository is a structured collection of my Capture The Flag (CTF) write-ups from various platforms such as picoCTF, HackTheBox, TryHackMe, and CTFtime competitions.

The main purpose of this repository is to document my learning process in cybersecurity and to keep a clear record of solved challenges, techniques, and tools used.

## Repository Structure

The write-ups are organized by platform and category to make navigation easier:

- **picoCTF** – Organized by challenge type (forensics, web exploitation, cryptography, reverse engineering, binary exploitation, etc.)
- **HackTheBox** – Categorized by topic such as web, forensics, reversing, and misc
- **TryHackMe** – Organized by difficulty level and rooms
- **tools-and-notes** – Personal notes, commands, and cheatsheets

## Write-up Format

Each challenge write-up typically includes:

- Challenge description
- Initial analysis and approach
- Tools and commands used
- Step-by-step solution
- Final flag
- Key takeaways or lessons learned

## Goals

- Improve problem-solving skills in cybersecurity
- Build consistent documentation habits
- Track progress across different CTF platforms
- Create a personal reference for future challenges

## Disclaimer

This repository is for educational purposes only. All solutions are based on legal CTF challenges and intentionally vulnerable environments.
